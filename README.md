# 丸洋建設 適性診断クイズ

このリポジトリは丸洋建設の採用活動向けに作成された職業適性診断クイズアプリです。

## 概要

建設業に興味のある方向けに、簡単な質問に答えることで建設業への適性を診断できるウェブアプリケーションです。5つの質問に答えることで、あなたの建設業への適性度が星評価で表示されます。

## 技術スタック

- React 18
- Vite
- TailwindCSS
- Framer Motion（アニメーション）

## 機能

- 5つの質問からなる建設業適性診断クイズ
- インタラクティブなUI/UXデザイン
- 結果画面での適性度表示（星評価）
- SNSシェア機能（X、LINE）
